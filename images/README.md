# Images Directory

This directory is for storing images used on the PetPal website.

## Recommended Images to Add

1. **App Icon** - `app-icon.png` (512x512px or larger)
2. **App Screenshots** - `screenshot-1.png`, `screenshot-2.png`, etc.
3. **Logo** - `logo.png` (transparent background recommended)
4. **Hero Image** - `hero-bg.jpg` (if you want a background image)
5. **Feature Icons** - Custom icons for features (optional)

## Image Guidelines

- **Format:** PNG for icons/logos (transparent background), JPG for photos
- **Optimization:** Compress images to reduce file size
- **Naming:** Use lowercase with hyphens (e.g., `app-screenshot-1.png`)
- **Size:** Keep images under 1MB when possible

## Usage in HTML

Once you add images, reference them in your HTML files like this:

```html
<img src="images/app-icon.png" alt="PetPal App Icon">
```

## Image Compression Tools

- [TinyPNG](https://tinypng.com/) - PNG/JPG compression
- [Squoosh](https://squoosh.app/) - Image optimization
- [ImageOptim](https://imageoptim.com/) - Mac app for compression
